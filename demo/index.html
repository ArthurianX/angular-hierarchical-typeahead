angular-hierarchical-typeahead (WIP)

Multi Purpose Expand / Collapse Component

Component API / Options
Option Value Description
art-typeahead The main directive declaration
resizr-type string Accepted values are 'zoom' and 'css'. 'zoom' uses the zoom css property while 'css' just resizes the container
resizr-collapsed boolean When 'true', the container is collapsed to begin with.
resizr-callback function If present, this will callback a function inside your controller
resizr-appear-hover boolean When true, the controls are hidden and will appear on hover
resizr-ratio string Default is `50/50`, the ratio with which the container will be resized [width]/[height]
resizr-position string Position of the controls button: "bottom-left", "bottom-right", "right", "left", "top-left", "top-right"
resizr-border string Values accepted are colors, when present the container will have a border with the specified color, the controls will have the same colors.
resizr-parent-class string Name of class to be replaced
resizr-parent-level integer # of parents to go up for the `resizr-parent-class` property, default is `0`
resizr-adjacent boolean If enabled all the sibling at the requested parent level will be resized, with a class if present or with css
