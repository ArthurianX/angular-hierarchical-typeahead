Inline year selector with range support

Demo
Directive Demo

Outside attachment

Delayed & triggered attachment will happen here.

Options
Option Value Description
years integer The number of years the selector should have. Starting from the current year down.
model scope variable The model that's gonna updated on a new selection. It's needs to be read as a string so make sure it's a string (model: 'myScopeVar')
attachTo class/#/tag If we're going to attach the selector to another element we need to supply a selector (with it's punctuation).
attachNow scope variable This is the listener that the directive will trigger the deferred attachment from.
externalCallback fn Function used to do callback in the current controller's scope
range bool Enables selection of ranges.
drag bool Enables the dragging of the selected active year to change the selection or the range.
